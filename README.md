# iTunesmasterillumination
## これなに？
  - iTunesでアイマス曲を流すとアーティスト情報から画面に色を出してくれるだけのアプリ
  - mac専用
  - node.js + express + socket.io
  - nodeの勉強に作った
  - ネーミングセンス

## 使い方
  - npm install
  - node bin/www

## ライセンス
  - カラーコードはアイマスDBの[キャラクターAPI](http://api.imas-db.jp/character.html)を使わせて頂きました
    - [THE IDOLM@STER データベース](http://imas-db.jp/)
